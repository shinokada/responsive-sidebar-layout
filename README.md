# Responsive Sidebar Layout

## Creating a project

```bash
# clone this repo
git clone git@github.com:shinokada/responsive-sidebar-layout.git

# install dependencies
cd responsive-sidebar-layout
npm i 
# or using pnmp
pnpm i

# run the local serve
npm run dev
```
